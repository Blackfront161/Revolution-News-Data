Privacy Policy

Last updated: June 22, 2026

This privacy policy governs the mobile application World Revolution News. Our core architecture is built around absolute anonymity and zero user-tracking.

1. Data Collection & Processing

World Revolution News does not collect, store, harvest, or share any personal identifiable information (PII).

  • No Accounts: You do not need to register or create an account to use this application.
  • No Telemetry: There are no background analytic suites, tracking cookies, or diagnostic logs active.
  • Local Storage: Bookmarks, system language choices, and read histories are stored exclusively on your local device's memory and never leave it.

2. Third-Party Services & Translation API

To deliver automated translations, requests are routed exclusively via a protected, independent server infrastructure (Proxy). Your real IP address or hardware metadata is never passed to corporate API entities (Google Gemini).

3. Network Security & Anonymity

This software is completely transparent and fully compatible with VPN clients and network anonymizers such as Tor/Orbot. We structurally encourage the use of security mechanisms to secure your data pipeline.

4. Changes to This Policy

We preserve the right to adjust this document to reflect structural improvements. Any changes will be instantly accessible inside this public URL repository.
